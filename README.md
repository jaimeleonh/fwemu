
## Firmware - Emulator Madrid Setup

### Download
```
git clone https://github.com/jaimeleonh/fwemu.git
```

### Updates
- Inside fwemu
```
git remote -v
```
- Check that it appears:  
```
origin	https://github.com/jaimeleonh/fwemu.git (fetch)
origin	https://github.com/jaimeleonh/fwemu.git (push)
```

- If so:
```
git pull origin 
```
